Virtual Environment:

virtual Environment is used to store or install required modules to store temporarly to avoid interaction between permanent files and temporary files.

To set Virtual Environment we use following command in terminal and that is Python -m venv .env.

To activate Virtual Enviroment we use .\.env\Scripts\Activate.ps1.

If error occurs then we need to open terminal(Not of VSCode) and run the command i.e Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted. Need to enter this command in any terminal if you are on Windows 11. If you are on below Windows 11 you can use powershell terminal only.
